## 一、项目概述

### 1.1 项目名称
**PosePal（拍照搭子）—— 你的 AI 拍照搭子**

### 1.2 一句话介绍（提交用，≤120 字）
PosePal 是拍照前的 AI 搭子：拍一张现场环境照，AI 就看懂光线与背景，给出适合你的姿势和取景技巧卡片，让每个女生不靠摄影师也能轻松拍出好看的照片，帮女性把"拍好看"从反复试错里夺回来。

### 1.3 项目定位
拍照前的 AI 摄影指导工具。不是美颜、不是滤镜、不是后期，而是**在按下快门之前**给用户可照做的拍摄方案。

### 1.4 命题赛道
**好看时间**：女性花在"让自己好看"上的时间。现实里这段时间常被"不会摆、拍不好、反复重拍"的挫败感吞掉。PosePal 让女性在按下快门前就拿到可照做的方案，把"好看时间"从焦虑试错变成轻松出片。

### 1.5 AI 创新点
1. **多模态"看懂现场"**：AI 真正读懂环境照的光线方向、背景元素、空间结构，给因地制宜的建议
2. **场景 → 姿势 → 技巧 的结构化推理链**，输出可照做卡片
3. **图像模型把推荐姿势可视化**（加分项），而非纯文字描述

## 二、背景与问题

### 2.1 解决的问题
女性想拍好看的照片，卡点不在设备，而在"到了现场不知道怎么摆、怎么站、怎么取景"：

- 不会摆姿势，站着僵硬，拍几十张挑不出一张
- 不懂利用现场环境（光线、背景、前景、线条）
- 身边没有懂拍照的搭子，出片全靠运气
- 现有工具（美颜 / 滤镜 / 后期）解决的是"拍完之后"，真正的空白在"按下快门之前"

### 2.2 解决方案
用户在拍照前，用手机拍一张现场环境照片上传，AI 完成三件事：
1. **场景理解**：识别环境类型、光线方向、可用背景与前景元素
2. **姿势匹配**：推荐 3-5 个可直接模仿的姿势（附示意图），说明站位、手部动作、朝向
3. **拍照技巧**：给出取景建议（机位高低、角度、构图法）、光线利用、避坑提示

输出以**卡片式、可照做**的形式呈现，降低理解成本——用户照着摆即可出片。

## 五、用户流程

### 5.1 主链路

```
首页 → 上传页 → 分析中 → 结果页
```

**详细步骤：**

1. **首页**：看到"拍前见你的 AI 摄影师"，点击"Start"
2. **上传页**：
   - 选"Scene photo"或"Selfie mode"
   - 上传自己的照片，或点 4 个示例场景之一
   - 点击"Direct my shot →"
3. **分析中**：等待约 20 秒，AI 读光线、找角度、选姿势
4. **结果页**：
   - 大图叠加白色人形线稿（AR 示意）
   - 底部 3 个姿势 tab，点击切换
   - 右上角 Tips 展开光线分析、最佳拍摄点
   - 底部"New photo"重新开始

### 5.2 关键交互

| 交互 | 说明 |
|------|------|
| 上传照片 | 支持 drag & drop 和点击选择 |
| 示例场景 | 一键加载，降低首次体验门槛 |
| 姿势切换 | 点击 tab 秒级切换，线稿模板即时加载 |
| Tips 面板 | 展开查看光线方向、光质、最佳拍摄点、构图提示 |
