### 1.2 光线状态分级

| 状态 | 含义 | 推荐行为 |
|------|------|---------|
| 可拍 ✅ | 光线适合拍摄 | 直接推荐姿势 |
| 注意 ⚠️ | 光线需调整 | 先输出调整建议，再推荐姿势 |
| 避雷 ❌ | 光线不适合 | 不推荐姿势，只输出调整建议 |

### 1.4 AI 光线判断逻辑

```
IF 人脸阴影在眼窝/鼻下深 → 顶光 → 提示"换到窗边"
ELSE IF 人脸暗、背景过曝 → 逆光 → 提示"侧身45°"
ELSE IF 半张脸亮半张暗 → 侧光 → 提示"亮面朝镜头，直接拍"
ELSE IF 全脸均匀亮 → 顺光 → 提示"清新风，小心眯眼"
ELSE IF 发丝发光+脸不黑 → 侧逆光 → 提示"氛围感绝了"
ELSE → 默认推荐"前侧光站位"
```

## 二、场景识别模块

### 2.1 核心场景（3 个）

| 场景 | 识别依据 | 构图特点 |
|------|---------|---------|
| 咖啡厅/餐桌前 | 桌面、杯子、窗户、吊灯 | 栏杆/网格线条 |
| 室内看展/书店 | 门框、拱门、书架、展品 | 框架结构、背景杂乱 |
| 户外/草地/公园 | 草地、树木、开阔天空 | 天空背景 |

### 2.2 AI 识别维度（每场景 3 个）

**维度 1：场景类型**
- 判断依据：识别到的物体
- 输出：场景名称

**维度 2：光线方向**
- 判断依据：明暗分布特征
- 输出：光线类型 + 状态 + 建议

**维度 3：构图分析**
- 判断依据：背景线条/结构/杂乱程度
- 输出：构图建议

### 2.3 构图提示规则（3 种）

| 情况 | 判断依据 | 建议 |
|------|---------|------|
| 框架结构 | 识别到门框/书架缝隙/拱形结构 | "检测到框架结构，可用作天然画框，人站中间有故事感。" |
| 背景杂乱 | 识别到多人/杂乱展品 | "背景较杂乱，建议低机位仰拍，用干净墙面或天空做背景。" |
| 栏杆/网格 | 识别到窗户栏杆/网格线条 | "背景有栏杆，建议手机举高俯拍避开杂乱线条。" |

## 三、姿势库模块（27 个姿势）

### 3.1 姿势数据结构（统一 6 字段）

每个姿势包含：
1. **姿势名**
2. **适合光线**
3. **虚线动作轨迹**（步骤化，4-5 步）
4. **朋友照着念的口令**（完整口语化，可直接念给被拍者）
5. **表情指导**（嘴/眼/下巴/整体表情，细分到角度）
6. **避坑提示**（一句话）

### 3.2 姿势类型分类（4 种）

| 类型 | 说明 |
|------|------|
| 全身 | 人物全身入镜 |
| 半身 | 腰部以上入镜 |
| 背影 | 背对镜头 |
| 动态 | 行走中抓拍 |

### 3.3 场景一：餐桌前（9 个姿势，坐姿为主）

#### 姿势 1：拿杯看窗外
- **适合光线**：侧光
- **虚线轨迹**：正坐，手放桌上，脸正对前方 → 脸慢慢转向窗户方向，到侧脸 45° 停 → 右手抬起，握住杯把 → 杯子凑到嘴边（杯口贴下唇下方，不挡嘴）→ 眼神从看杯子→慢慢看向窗外远处
- **口令**："脸往窗户那边慢慢转，停。右手去拿杯子，抬起来凑到嘴边，先不喝。眼睛从杯子慢慢往窗外看出去，像在想事情。好，拍了。"
- **表情**：嘴巴放松微闭，不要用力抿；眼神看向窗外远处，放空感，带一点点淡淡笑意；下巴微抬 10°，显下颌线
- **避坑**：杯子别挡住嘴，放嘴唇下方一掌宽位置

#### 姿势 2：双手捧杯微笑
- **适合光线**：柔光、侧光
- **虚线轨迹**：坐直，双手放腿上 → 上半身往前倾 15°，胸口离桌边一拳 → 双手抬起，从两侧捧住杯子 → 下巴微微下沉，搁在杯口上方 → 抬头看镜头，嘴角上扬
- **口令**："坐直。上半身往桌子这边倒一点，停。手抬起来捧住杯子。下巴放杯口上面，抬头看我。笑，不用露齿，嘴角上提就好。拍了。"
- **表情**：微笑，嘴角往两边提，不露齿；眼神看镜头，眼睛微微弯成月牙形；下巴微收，搁在杯口，低头然后抬眼看镜头
- **避坑**：别耸肩，肩膀往下沉；笑的时候别眯眼过度

#### 姿势 3：托腮看镜头
- **适合光线**：柔光、侧光
- **虚线轨迹**：坐直，手放桌面 → 右手抬起，手肘撑桌面 → 手掌托住右侧下巴（轻放，不用力）→ 脸从正对→往右微转 15° → 眼神从平视→看镜头
- **口令**："右手抬起来，手肘撑桌上，手掌托住右边下巴。脸往右转一点点，像要把脸轻轻搁在手上。看我，笑。拍了。"
- **表情**：微笑或微张，自然放松；眼神直视镜头，眼神柔和，带一点俏皮感；眉毛自然上抬一点点，显得有精神
- **避坑**：手掌轻轻托，不要用力挤脸肉；手肘撑稳别晃

#### 姿势 4：回眸抓拍
- **适合光线**：逆光（侧身吃光）、侧光
- **虚线轨迹**：侧身坐，脸朝窗户，举手机假装拍窗外 → 听到口令，脖子带动脸慢慢向左转 → 脸转到一半，看向镜头 → 定住，眼神锁定镜头
- **口令**："侧身坐，脸朝窗户，举手机假装拍外面。我喊一二三你慢慢回头看我。一、二、三——回。慢一点，头不用转到底，看我就行。好，保持，拍了。"
- **表情**：嘴巴微张，显得自然惊讶感；眼神回头看镜头，带一点"咦？"的意外表情；整体表情抓拍感，不要太刻意
- **避坑**：回头时脖子往前伸，不要缩着；眼神别翻白眼

#### 姿势 5：手持餐具看镜头
- **适合光线**：柔光
- **虚线轨迹**：坐直，正对餐桌 → 右手拿叉/勺，叉一块食物 → 举到嘴边但先不吃 → 抬头看镜头，俏皮表情 → 张嘴做"啊"状
- **口令**："拿叉子叉一块蛋糕，举到嘴边。抬头看我，张嘴做'啊'的样子，像要喂我。别真吃啊，定住。拍了。"
- **表情**：嘴巴张成"啊"形，不要太大；眼神看镜头，带调皮感；整体表情可爱俏皮
- **避坑**：食物别戳破，保持完整；嘴巴不要张太大

#### 姿势 6：双手交叠趴桌上
- **适合光线**：柔光、侧光
- **虚线轨迹**：坐直，正对桌子 → 上半身往前趴，胸口贴桌边 → 双手交叠叠在桌上（手腕叠手腕）→ 下巴搁在手臂上 → 微微歪头看镜头
- **口令**："往前趴下来，双手叠在桌上。下巴搁手上。头往左边歪一点点，看我，微笑。拍了。"
- **表情**：微笑，带一点点慵懒感；眼神从下往上看镜头，温柔感；整体表情像午后晒太阳的猫，慵懒放松
- **避坑**：肩膀放松，不要耸肩；歪头角度别太大，15° 就够

#### 姿势 7：侧脸不笑看窗外（清冷感）
- **适合光线**：侧光、逆光（侧身吃光）
- **虚线轨迹**：正坐，脸正对前方 → 身体和脸同时转向窗户（侧脸 90°）→ 手轻握杯子放桌上 → 眼神看向窗外远处 → 面无表情，安静感
- **口令**："整个人转向窗户，侧脸对着我。手握着杯子放桌上就好。看窗外，不要笑，放空。好，拍了。"
- **表情**：嘴巴闭合，自然放松，不笑；眼神看向远处，放空感，带一点点疏离；整体表情清冷、安静、故事感
- **避坑**：不要瞪眼，眼神柔和放空；肩膀别绷着

#### 姿势 8：撩头发看窗外
- **适合光线**：侧光
- **虚线轨迹**：正坐，脸微侧 → 右手从桌面抬起 → 手指插入发根，轻轻撩起头发 → 手停在耳朵旁边 → 脸顺势转向窗户方向 → 眼神看窗外
- **口令**："右手抬起来，撩一下头发，手停在耳朵旁边。脸顺着手的动作转向窗户。看窗外，不要看我。拍了。"
- **表情**：嘴巴自然闭合，带一丝若有若无的笑意；眼神看向窗外，柔和的；整体表情自然、随性
- **避坑**：撩头发时手指不要抓太紧，轻轻带过就好

#### 姿势 9：低头切食物/看桌面被抓拍
- **适合光线**：柔光、侧光
- **虚线轨迹**：坐直，拿刀叉在切食物 → 低头专注切 → 听到口令，停住动作 → 保持低头，但眼神抬起来看镜头 → 定格
- **口令**："假装在切牛排/蛋糕，低头看着盘子。好，停，保持低头，但眼睛抬起来看我。对，像偷瞄我那样。拍了。"
- **表情**：嘴巴自然放松，或带一点被发现的浅笑；眼神从下往上看镜头，像"偷瞄被抓到"的感觉；整体表情俏皮、害羞、自然的被抓拍感
- **避坑**：头不要抬起来，保持低头，只抬眼睛；动作停住不要晃

### 3.4 场景二：看展/书店（9 个姿势）

#### 姿势 1：侧身看展品（全身）
- **适合光线**：侧光、柔光
- **虚线轨迹**：站在展品前，正对展品 → 身体转向右侧，侧身对展品（脚打开与肩同宽）→ 双手抱胸或手自然垂放 → 脸侧向 45° 看展品 → 定住，不看镜头
- **口令**："站在画前面，身体转向右边，侧身对着我。手抱胸，脸看画，别看我。好，保持，拍了。"
- **表情**：嘴巴闭合，自然放松；眼神看展品，专注感；整体表情沉浸在欣赏中的安静感
- **避坑**：身体不要正对展品，侧身显薄；脚尖朝右，脸朝画

#### 姿势 2：背影看展品（背影）
- **适合光线**：所有光线
- **虚线轨迹**：站在展品前，正对展品 → 直接背对镜头 → 双手背在身后或自然垂放 → 头微微侧向右边，露出一点点侧脸轮廓 → 定住，完全不看镜头
- **口令**："站到画前面，背对着我。手背到身后，头往右边转一点点，让我看到你一点点侧脸就行。别回头，保持。拍了。"
- **表情**：嘴巴看不到，不用管；整体氛围安静、神秘、故事感；侧脸轮廓下巴微抬，显侧脸线条
- **避坑**：不要真的回头，会变成回眸不是背影；头顶别被展品切到

#### 姿势 3：回眸看镜头（全身）
- **适合光线**：侧光、柔光
- **虚线轨迹**：背对镜头站在展品前 → 脚不动，上半身从腰部以上转身 → 脸转向镜头 → 手自然垂放或拿导览册 → 眼神锁定镜头
- **口令**："站到画前面，背对我。好，上半身转回来看我，脚别动。手拿导览册放身前。脸转过来，看我笑。拍了。"
- **表情**：嘴巴微笑，像被熟人叫住；眼神直视镜头，带一点惊喜感；整体表情温暖、自然
- **避坑**：只转上半身，脚保持不动

#### 姿势 4：手摸展品柜（半身）
- **适合光线**：侧光、柔光
- **虚线轨迹**：站在展柜前，正对展柜 → 身体微侧 15° → 右手抬起，手指轻轻放在展柜玻璃上 → 头侧向 45° 看镜头 → 表情温柔，定住
- **口令**："站到展柜前面，身体侧一点点。右手放玻璃上，手指张开。脸转过来看我，微笑。好，拍了。"
- **表情**：嘴巴柔和微笑；眼神看镜头，温柔感；整体表情优雅、安静、有质感
- **避坑**：手不要用力压玻璃，指尖轻放就好

#### 姿势 5：扶墙看镜头（半身）
- **适合光线**：侧光、柔光
- **虚线轨迹**：站在干净墙面前（不要有展品）→ 身体正对镜头，肩膀放松 → 右手抬起，手掌贴墙，高度在肩膀位置 → 左手自然下垂或拿手机 → 脸正对镜头，微笑
- **口令**："站到墙前面，正对我。右手抬起来，手掌贴墙上，高度在肩膀。左手拿手机自然放着。看我，笑。拍了。"
- **表情**：嘴巴露齿笑或微笑；眼神直视镜头，有神；整体表情清爽、大方
- **避坑**：手掌平贴墙，不要抓墙；肩膀下沉放松

#### 姿势 6：走路抓拍（全身、动态）
- **适合光线**：柔光
- **虚线轨迹**：离镜头 5 步远，站在走廊一端 → 朝镜头方向慢慢走 → 视线看前方或看展品 → 走到镜头前 3 步时，转头看镜头 → 继续走，抓拍
- **口令**："从那边走廊走过来，走慢一点。眼睛看前面。走到第三个地砖的时候转头看我一下，继续走，别停。我连拍。走。"
- **表情**：嘴巴自然放松；眼神看前方→转头瞬间看镜头；整体表情动态抓拍感
- **避坑**：脚步不停，走慢；转头自然，不要甩头

#### 姿势 7：背影手摸画框（背影）
- **适合光线**：所有光线
- **虚线轨迹**：站在画作前，背对镜头 → 双手抬起，手指轻触画框下沿 → 头微低，看着画 → 完全背影，不露脸
- **口令**："站到画前面，背对我。双手抬起来，手指摸画框下沿。头低一点看画。别回头。拍了。"
- **表情**：看不到脸，靠身体语言；手指轻触画框，有互动感；整体氛围文艺、安静
- **避坑**：手不要举太高，摸画框下沿就好；站直别驼背

#### 姿势 8：假装拍照（半身）
- **适合光线**：所有光线
- **虚线轨迹**：站在展品前，正对展品 → 举起手机假装拍展品 → 脸转向镜头，俏皮表情 → 定格
- **口令**："站到展品前面，举手机假装拍它。然后脸转过来看我，俏皮一点。好，定住。拍了。"
- **表情**：嘴巴俏皮笑，微嘟嘴；眼神看镜头，带调皮感；整体表情活泼、可爱
- **避坑**：手机不要挡脸，举到眼睛高度

#### 姿势 9：蹲下看低处展品（全身）
- **适合光线**：侧光、柔光
- **虚线轨迹**：站着面对低处展品/展柜 → 单膝蹲下（或双脚并拢蹲下）→ 手撑膝盖或摸展柜 → 脸转向镜头微笑 → 全身展现在画面里
- **口令**："蹲下来，面对展柜。一只手撑膝盖上。脸转过来看我，笑。好，拍了。"
- **表情**：嘴巴微笑或露齿笑；眼神从下往上看镜头（显高）；整体表情可爱、轻松
- **避坑**：蹲姿要优雅，膝盖并拢或前后错开；不要正对镜头蹲

### 3.5 场景三：户外/草地（9 个姿势）

#### 姿势 1：逆光侧身回头（全身）
- **适合光线**：逆光（最佳）、侧逆光
- **虚线轨迹**：背对太阳站立，正对镜头方向 → 身体转向右侧 45°（让光从背后打过来）→ 听到口令，从腰部以上转身回头看镜头 → 脚不动，只转上半身 → 发丝被光照亮，定住
- **口令**："背对太阳站，身体侧 45°。我喊一二三你回头看我，脚别动。一、二、三——回。头发被光照得发光了，好美，别动。拍了。"
- **表情**：嘴巴微张，带一点点惊喜感；眼神温柔看镜头，像被叫住回头看；整体表情柔和的惊喜感，眼神带笑意；头发逆光下发丝边缘会发光，效果拉满
- **避坑**：回头时脖子往前伸，别缩着；脸正对镜头方向吃光

#### 姿势 2：张开双臂迎风（全身）
- **适合光线**：逆光、侧逆光
- **虚线轨迹**：背对镜头站，面朝开阔方向 → 双手从身体两侧慢慢抬起，举到肩膀高度 → 手心朝下或朝前 → 头微仰，眼睛看天空 → 感受风，定住
- **口令**："背对我站到草地中间。双手慢慢张开，像要飞一样，举到肩膀高。头抬起来看天空。闭眼，感受风。好，拍了。"
- **表情**：嘴巴自然闭合，嘴角带一点点笑；眼神闭眼或看天空，享受感；整体表情自由、舒展、放松
- **避坑**：肩膀别耸起来，下沉；手臂伸直别弯

#### 姿势 3：侧身看镜头微笑（全身）
- **适合光线**：顺光、柔光
- **虚线轨迹**：正对镜头站 → 身体转向左侧 45° → 双脚前后错开（前脚脚尖朝前，后脚横放）→ 双手自然垂放或拿花/帽子 → 脸转回来看镜头微笑
- **口令**："正对我。身体往左边转 45°，脚一前一后站。手拿帽子放身前。脸转回来看我，笑。拍了。"
- **表情**：嘴巴微笑，露齿或不露齿都可；眼神直视镜头，温暖柔和；整体表情甜美、大方、清爽
- **避坑**：不要正对镜头站，侧身显薄显高

#### 姿势 4：蹲下摸草地/花（全身）
- **适合光线**：所有光线
- **虚线轨迹**：站在草地/花丛前 → 单膝蹲下或双脚并拢蹲下 → 一只手伸出去摸草地/花 → 另一只手搭在膝盖上 → 脸转向镜头微笑
- **口令**："蹲下来，一只手摸草地，另一只手放膝盖上。脸转过来看我，笑。好，拍了。"
- **表情**：嘴巴露齿笑或微笑；眼神看镜头，甜美；整体表情可爱、亲近自然
- **避坑**：蹲姿膝盖并拢或前后错开，避免走光；不要驼背

#### 姿势 5：背影看远方（背影）
- **适合光线**：所有光线（逆光更佳）
- **虚线轨迹**：背对镜头站立 → 双手自然垂放或背在身后 → 头微仰，看远方天空/日落 → 完全不看镜头 → 安静定住
- **口令**："站到那边，背对我。手背身后，抬头看远方。别回头，保持。好，拍了。"
- **表情**：看不到脸；身体语言站直不驼背；整体氛围安静、治愈、有故事感
- **避坑**：站直别驼背；头发别挡完脖子

#### 姿势 6：往前走回头抓拍（全身、动态）

- **口令**："从那边朝我走过来，走慢一点。眼睛看前面。好，走到第四个脚步的时候转头看我一下，别停，继续走。我连拍。走。"

## 六、AI 系统提示词（完整版）

### 6.1 角色设定

你是一个专业的摄影指导 AI，名为 PosePal。你的任务分两步：先诊断光线，再指导姿势。

### 6.2 工作流程

**第一步：场景+光线诊断**

用户上传一张环境照，识别：
- 场景类型：咖啡厅/看展/户外
- 光线类型：侧光/顶光/逆光/柔光/顺光
- 光线状态：可拍/避雷/注意
- 调整建议（如果光线不 OK）
- 构图提示（如适用）

判断标准：
- 侧光（脸一侧亮一侧暗）→ 可拍。话术："侧光显立体，亮面朝向镜头效果更佳。"
- 顶光（眼窝鼻下有深阴影）→ 避雷。话术："顶光让面部有阴影。建议移到窗边，让光线从侧面照过来。"
- 逆光（背景过曝脸偏暗）→ 注意。话术："逆光脸偏暗。建议侧身 45° 面向光源，让脸吃到光。"
- 柔光（无强烈阴影）→ 可拍。话术："柔光不挑角度，适合清新风，直接开拍。"
- 顺光（人脸亮且无阴影）→ 注意。话术："顺光脸虽亮但偏平，建议试试逆光拍发丝发光。"

构图提示：
- 框架结构 → "检测到框架结构，可用作天然画框，人站中间有故事感。"
- 背景杂乱 → "背景较杂乱，建议低机位仰拍，用干净墙面或天空做背景。"
- 栏杆/网格 → "背景有栏杆，建议手机举高俯拍避开杂乱线条。"

**第二步：姿势指导**（仅光线状态为"可拍"时输出）

根据场景类型和光线类型，从对应姿势库中推荐 3 个姿势。

匹配规则：
- 光线为侧光/柔光时：推荐该场景下适合这两种光线的姿势
- 光线为逆光时：先输出"逆光氛围感强，侧身 45° 让光打在侧面，发丝会发光"，再推荐逆光适合的姿势
- 光线为顺光时：推荐顺光/柔光适合的姿势

### 6.4 重要规则

- 如果光线状态为"避雷"，不推荐姿势，只输出光线诊断
- 如果光线状态为"注意"，先输出调整建议，再推荐姿势
- 如果光线状态为"可拍"，直接推荐 3 个姿势
- 姿势必须从对应场景的姿势库中选择，不要自己编
